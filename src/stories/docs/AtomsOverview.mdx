# Atoms

Atoms are the **smallest indivisible UI elements** in the design system. They have no custom dependencies and are highly reusable across the application.

## Atom Characteristics

| Characteristic | Description |
|---------------|-------------|
| **Independence** | No dependencies on other custom components |
| **Pure** | Presentational only, no business logic |
| **Reusable** | Used across multiple molecules and organisms |
| **Accessible** | Full keyboard and screen reader support |
| **Tested** | 100% test coverage required |

## Creating New Atoms

Each atom must have:

```
component-name/
├── component-name.tsx          # Component implementation
├── component-name.stories.tsx  # Storybook stories
├── component-name.test.tsx     # Unit tests
└── index.ts                    # Barrel export
```

See individual atom stories for detailed API documentation.
