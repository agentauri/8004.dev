# Color Palette

Agent Explorer uses a **NES/Sega inspired color palette** optimized for dark theme interfaces. All colors are available as CSS custom properties.

## Primary Colors

Action and accent colors inspired by classic 8-bit games.

<ColorGrid colors={getColorsByCategory('primary')} />

## Dark Theme Base

Grayscale palette for backgrounds, surfaces, and text.

<ColorGrid colors={getColorsByCategory('dark')} />

## Chain Colors

Each supported blockchain has a unique identifying color.

<ColorGrid colors={getColorsByCategory('chain')} />

## CRT Glow Effects

Semi-transparent colors for the signature CRT glow effects.

<ColorGrid colors={getColorsByCategory('glow')} />

## Usage Guidelines

### Primary Colors
- **Blue Sky** - Primary actions, links, active states
- **Red Fire** - Errors, destructive actions, Ethereum
- **Green Pipe** - Success states, positive indicators
- **Gold Coin** - Highlights, premium features, warnings

### Grayscale
- **Gray 200** - Primary text
- **Gray 400** - Secondary text, labels
- **Gray 600** - Borders, disabled states
- **Gray 800** - Elevated surfaces, cards

### Implementation

```css
/* Using CSS variables */
.button {
  background: var(--pixel-blue-sky);
  color: var(--pixel-black);
}

/* With glow effect */
.card:hover {
  box-shadow: 0 0 20px var(--glow-blue);
}
```

```tsx
// Using Tailwind classes
<button className="bg-[var(--pixel-blue-sky)] text-[var(--pixel-black)]">
  Click me
</button>
```
