# Design Tokens Overview

Agent Explorer's design system is built on a foundation of **design tokens** - the atomic building blocks that define our visual language. This 80s retro pixel art aesthetic is inspired by NES/Sega era games while maintaining enterprise-grade usability.

## Quick Reference

| Category | Count | Purpose |
|----------|-------|---------|
| Colors | 20 | Primary, grayscale, chain-specific, glow effects |
| Typography | 4 | Display, heading, body, UI fonts |
| Spacing | 11 | Consistent spacing based on 4px grid |
| Animations | 4 | Pixel-perfect transitions and effects |

## Typography

Our typography system uses four distinct font families for different use cases.

| Font | Variable | Usage |
|------|----------|-------|
| Press Start 2P | `--font-pixel-display` | Page titles, hero text |
| Silkscreen | `--font-pixel-heading` | Card titles, badges, labels |
| JetBrains Mono | `--font-pixel-body` | Descriptions, addresses, code |
| Inter | `--font-sans` | Form inputs, long text |

### Implementation

```tsx
// Display font for titles
<h1 className="font-[family-name:var(--font-pixel-display)]">
  AGENT EXPLORER
</h1>

// Heading font for section titles
<h2 className="font-[family-name:var(--font-pixel-heading)]">
  Featured Agents
</h2>

// Body font for descriptions
<p className="font-[family-name:var(--font-pixel-body)]">
  0x1234...5678
</p>

// UI font for forms
<input className="font-sans" />
```

## Spacing Scale

Our spacing system is based on a **4px grid** for consistent rhythm throughout the interface.

| Token | Value | Pixels | Usage |
|-------|-------|--------|-------|
| 0 | 0 | 0px | No spacing |
| 1 | 0.25rem | 4px | Tight spacing |
| 2 | 0.5rem | 8px | Default gap |
| 3 | 0.75rem | 12px | Card padding |
| 4 | 1rem | 16px | Section gap |
| 6 | 1.5rem | 24px | Large gap |
| 8 | 2rem | 32px | Section padding |
| 12 | 3rem | 48px | Page sections |
| 16 | 4rem | 64px | Major sections |

### Guidelines

- Use **multiples of 4px** for all spacing
- **gap-2** (8px) for tight layouts
- **gap-4** (16px) for standard layouts
- **p-4** to **p-6** for card padding
- **py-8** to **py-16** for section spacing

## Animations

Agent Explorer uses **pixel-perfect animations** that respect the retro aesthetic while providing smooth user feedback.

### Animation Principles

1. Use `steps()` timing for authentic 8-bit feel
2. Keep duration under **300ms** for interactions
3. Use **transform** and **opacity** only for performance
4. Respect `prefers-reduced-motion`

### Available Animations

| Name | Class | Duration | Description |
|------|-------|----------|-------------|
| Pulse Glow | `animate-pulse-glow` | 2s | Subtle pulsing glow for active states |
| Blink | `animate-blink` | 1s | 8-bit style cursor blink |
| Fade In | `animate-fade-in` | 150ms | Quick fade in for modals |
| Slide Up | `animate-slide-up` | 200ms | Slide up entrance |

### Implementation

```tsx
// Pulsing glow for active elements
<div className="animate-pulse-glow">Active Agent</div>

// Blinking cursor effect
<span className="animate-blink">_</span>

// Modal entrance
<dialog className="animate-fade-in">
  Content
</dialog>

// Slide up notification
<div className="animate-slide-up">
  New agent registered!
</div>
```

### Reduced Motion Support

```css
@media (prefers-reduced-motion: reduce) {
  .animate-pulse-glow,
  .animate-blink {
    animation: none;
  }
}
```
