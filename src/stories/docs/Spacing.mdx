# Spacing

Agent Explorer uses a **4px base grid** for consistent spacing throughout the interface.

## Spacing Scale

<SpacingScale spacings={SPACING_TOKENS} />

## Usage Guidelines

### Component Padding

| Component Type | Padding |
|---------------|---------|
| Small buttons | `px-2 py-1` (8px, 4px) |
| Regular buttons | `px-3 py-1.5` (12px, 6px) |
| Large buttons | `px-4 py-2` (16px, 8px) |
| Cards | `p-4` (16px) |
| Sections | `p-6` (24px) |

### Gap Spacing

| Context | Gap |
|---------|-----|
| Inline elements | `gap-1` (4px) |
| Badge groups | `gap-2` (8px) |
| Card grids | `gap-4` (16px) |
| Page sections | `gap-6` (24px) |

### Margin Spacing

| Context | Margin |
|---------|--------|
| Between form elements | `space-y-2` (8px) |
| Between sections | `space-y-6` (24px) |
| Page margins | `mx-4` or `mx-6` |

## Implementation

```tsx
// Using Tailwind spacing utilities
<div className="p-4 space-y-2">
  <div className="flex gap-2">
    {/* content */}
  </div>
</div>

// Custom pixel-perfect spacing
<div className="p-[12px] gap-[6px]">
  {/* when you need exact pixels */}
</div>
```

## Grid System

The layout uses a flexible grid with consistent gutters:

```tsx
// 3-column grid with 16px gaps
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  {/* cards */}
</div>
```
